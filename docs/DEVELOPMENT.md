## Feature Requests

### High Priority

- [ ] **Custom Domain:** Configure refidao.com
- [ ] **Docs Subdomain:** Set up docs.refidao.com
- [ ] **SEO Optimization:** Meta tags, structured data, sitemap
- [ ] **Performance:** Image optimization, lazy loading

### Medium Priority

- [ ] **Newsletter Signup:** Integration with email service
- [ ] **Node Application Form:** Embedded or linked form
- [ ] **Search:** Site-wide search functionality
- [ ] **Analytics:** Privacy-respecting analytics (Plausible/Fathom)

### Low Priority / Future

- [ ] **Multi-language Support:** i18n for global audience
- [ ] **Member Dashboard:** Authenticated member area
- [ ] **Blog Integration:** Embedded blog posts from Ghost
- [ ] **Event Calendar:** Upcoming events display
